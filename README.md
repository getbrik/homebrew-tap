This is the official Homebrew tap for [Brik](https://github.com/getbrik/brik), the portable pipeline standard.

## Installation

```bash
brew tap getbrik/tap
brew install brik
```

Or install directly:

```bash
brew install getbrik/tap/brik
```

## Upgrade

```bash
brew update && brew upgrade brik
```

## Uninstall

```bash
brew uninstall brik
brew untap getbrik/tap  # optional: remove the tap
```
